# MANIFEST LEGISLACJI INŻYNIERYJNEJ (Code is Law)

## 1. Aksjomat Binarnej Prawdy
W erze cyfrowej prawo nie może być literaturą piękną podlegającą interpretacji. Prawo musi być algorytmem. Podmiot jest albo uczciwy (Stan 1), albo nie (Stan 0). Nie ma stanów pośrednich w bazach danych.

## 2. Prewencja zamiast Represji
Tradycyjne państwo karze po fakcie. CSW uniemożliwia popełnienie przestępstwa technicznie. Nie ścigamy złodziei – montujemy cyfrowe zamki, których nie da się otworzyć bez klucza walidacyjnego.

## 3. Transparentność Kernelu
Kod sterujący gospodarką musi być Jawny (Open Source). Każdy obywatel ma prawo wiedzieć, dlaczego system podjął decyzję o blokadzie. Ukrywanie algorytmów to domena systemów totalitarnych. Transparentność buduje zaufanie.

## 4. Matematyczna Nieuchronność
Zamiast armii urzędników – automatyzacja. Zamiast uznaniowości – instrukcja warunkowa `if/else`. Sprawiedliwość jest funkcją matematyczną, a nie polityczną.
