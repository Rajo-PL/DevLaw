# Program Bug Bounty (Nagrody za Błędy)

W systemie DevLaw "luka prawna" to błąd techniczny (*bug*).

1. **Zgłoszenie:** Obywatel ("White Hat") wysyła Raport Techniczny przez platformę, opisując błąd logiczny (np. pętla podatkowa) [38, 39].
2. **Klasyfikacja:** Low / Medium / Critical (np. wyciek kapitału) [39].
3. **Nagroda:** Wypłacana automatycznie przez Smart Contract, proporcjonalnie do zaoszczędzonych środków państwa [40].
4. **Naprawa:** Inżynierowie wdrażają "Patch". Historia zmian (Changelog) jest jawna [41].
